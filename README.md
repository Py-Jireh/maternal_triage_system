# Maternal Triage System

📌 Overview
A simple Python program to store patient information and assess maternal risk based on age, blood pressure, and weeks pregnant. High-risk patients are flagged automatically.

📌 Core Functionality
- Add new patient records (name, age, BP, weeks pregnant)
- Display all patients with risk status
- Count total patients in the system
- Highlight high-risk patients

📌 Technical Implementation
- Language: Python 3
- Data structures: Lists and dictionaries
- Control flow: if statements and loops
- Interactive command-line program
