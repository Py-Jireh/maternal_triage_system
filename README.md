Maternal Risk Assessment System
Optimizing Clinical Outcomes through Automated Patient Screening

📌 Overview 

A Python-driven diagnostic tool designed to categorize maternal health risks by analyzing blood pressure, age, and gestational milestones. 
This system transitions clinical workflows from manual sorting to data-informed prioritization, ensuring high-acuity cases receive immediate intervention.

📌Core Functionality

Centralized Records: Manages patient vitals and longitudinal health data.
Automated Triage: Instantly stratifies patients into risk tiers based on clinical thresholds.
Decision Support: Provides evidence-based recommendations for immediate nursing actions.

📌Technical Implementation

Language: Python 3
Logic: Utilizing dictionaries for data management and control flow for risk-level evaluation.
